Getting started with the leafnode-2 DARCS repository

Checking out the repository

To obtain the initial copy, type:

darcs get --partial \
http://home.pages.de/~mandree/leafnode/beta/darcs/ leafnode-2-darcs

The --partial only obtains the patches since the last checkpoint. Omit if you need to review very old patches.

Doing your first build

Auto-generated files are not part of the repository, so you must generate them before your first build.

  1. To generate configure and friends, type: 
    autoreconf -iv
  2. To configure and build leafnode, type: 
    mkdir -p build
cd build
../configure -C
make
  3. Then install as usual.

Updating your repository

Switch to your working directory, then type:

darcs pull

Submitting patches

To submit your modifications, send GnuPG-signed mail, or expose your repository and send the URL to pull from. See

darcs send --help
for details.
